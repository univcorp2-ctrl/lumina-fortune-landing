# Market research — 2026-07-07

「売れている」を第三者の推定売上だけで断定せず、現在利用されている大手・著名サービスの公式ページと公式ストア表示から、継続利用と課金に関係するプロダクトパターンを調べました。

## LINE占い

公式 Google Play 表示では、ビデオ対面、チャット、電話、メッセージの4手段、相談カテゴリーと占術の豊富さ、初回無料価値を前面に出しています。成功パターンは「今すぐ相談できる」「方法を選べる」「最初の不安を無料枠で下げる」です。

- https://play.google.com/store/apps/details?hl=ja&id=jp.naver.linefortune.android
- https://fortune.line.me/talk

## ココナラ占い

公式カテゴリは多数の出品から価格・評価・提供方法を比較でき、テキスト、電話など状況に合う相談方法を選べます。成功パターンは「選択肢」「比較可能性」「匿名レビュー」「購入前の理解」です。

- https://coconala.com/categories/3

## cocoloni占い館

公式ページは無料の12星座・タロットを十分に提供し、その先に低価格の特別占いと本格鑑定を配置しています。成功パターンは「先に無料で満足を作る」「無料結果と有料詳細が自然につながる」です。

- https://honkaku-uranai.jp/

## Co–Star

公式サイトと FAQ は、出生情報によるパーソナライズ、日々の更新、友人との相性、追加質問・高度な読み解きへのフリーミアム課金を説明しています。成功パターンは「日替わり習慣」「自分専用感」「質問の深掘り」「購入内容を後から再閲覧できること」です。

- https://www.costarastrology.com/
- https://www.costarastrology.com/faq

## LUMINA に採用したもの

| パターン | 実装 |
| --- | --- |
| 登録前の即時価値 | LP内で1分の無料鑑定が完結 |
| 悩み別の入口 | 恋愛・仕事・金運・自分探し |
| パーソナライズ | 日付、テーマ、任意プロフィールから決定的に生成 |
| 毎日戻る理由 | 東京日付で変わるカード、連続日数 |
| 無料から有料への連続性 | 無料結果の直下に詳細項目をプレビュー |
| 信頼 | 外部送信なし、料金例、注意事項、リセット |
| 再閲覧 | localStorage に履歴を保持する基盤 |

## 採用しなかったもの

- 恐怖、不幸、別れを断定して購入を迫るコピー
- 偽のカウントダウン、残席、架空レビュー、架空会員数
- 無料結果を見せる前の強制会員登録
- 解約や自動更新を隠す料金表示
- 医療・法律・金融判断を占いだけで誘導する表現
